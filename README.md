📝 Formulário HTML – Pesquisa Online

Projeto desenvolvido como exercício prático do curso da Alura, com foco na construção estruturada de formulários utilizando HTML5.

Embora seja um projeto acadêmico, ele demonstra fundamentos essenciais para desenvolvimento Front-End e aplicações web.

🎯 Objetivo

Construir um formulário funcional aplicando:

Estrutura semântica

Organização lógica de dados

Validação nativa do HTML

Boas práticas de usabilidade

🛠️ Tecnologias Utilizadas

HTML5

Validações nativas (required, pattern, min, max)

Tipos variados de input (text, number, date, email, tel, file)

💡 Competências Demonstradas

Organização e clareza estrutural

Atenção a detalhes e regras de validação

Aplicação de boas práticas sem uso de JavaScript

Estrutura preparada para integração futura com backend

Pensamento voltado para experiência do usuário

📌 Diferencial Técnico

Validação de telefone com expressão regular:

```<input type="tel" pattern="[0-9]{11}" required>```

Controle de faixa etária:

```<input type="number" min="12" max="100" required>```

🚀 Próximos Passos

Aplicação de CSS para melhoria visual

Implementação de validação dinâmica com JavaScript

Integração com backend para persistência de dados

Melhoria de acessibilidade com associação label e id

👨‍💻 Perfil do Desenvolvedor

Desenvolvedor em formação, com foco em fundamentos sólidos de HTML, organização de código e construção estruturada de interfaces.

Buscando evoluir para projetos completos com integração Front-End e Back-End.
